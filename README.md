# Api para geração e requisições de exames laboratoriais.

## Funcionalidade:

 - CRUD de exames e clientes.
 - Gerar solicitação de exames.
 - Calcular valor dos exames selecionados.
 - Impressão da solicitação em duas vias.

## Tecnologias
- Java 8
- Spring Framework
  - Spring Boot
  - Spring Validation
  - Spring Web
  - Spring Data Jpa
  - Spring Security
- Postgresql
- Lombok
